The Good Way
------------

First start with makeing an instance of the Container/Folder

  >>> root = getRootFolder()

  >>> from uvcsite.tests.fixtures.simpleaddon import AdressBook
  >>> adrbook = AdressBook()
  >>> adrbook 
  <uvcsite.tests.fixtures.simpleaddon.AdressBook object at ...>

  >>> import grok
  >>> class MyView(grok.View):
  ...     grok.context(AdressBook)
  ...     def render(self):
  ...         return "Yes"

  >>> from grok.testing import grok_component
  >>> grok_component('MyView', MyView)
  True

  >>> root['book'] = adrbook
  >>> from zope.component import getMultiAdapter
  >>> from zope.publisher.browser import TestRequest


  >>> view = getMultiAdapter((adrbook, TestRequest()), name=u"myview")
  >>> print view.render()
  Yes
