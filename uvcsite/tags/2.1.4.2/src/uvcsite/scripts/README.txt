ZODB-Management Scripts
-----------------------

Diese Scripts können mit:

 - bin/uvcsite-debug parts/omelette/uvcsite/scripts/listHomeFolder.py 
 - bin/uvcsite-debug parts/omelette/uvcsite/scripts/delObject.py

aufgerufen werden. 

Die Parameter für den Benutzer bzw. ID' des zu löschenden Objekts 
werden momentan direkt als Konstante in die Scripts eingetragen.
