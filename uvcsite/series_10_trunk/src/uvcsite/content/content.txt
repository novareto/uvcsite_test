Content
=======

:doctest:
:layer: uvcsite.tests.FunctionalLayer

Setup
-----
First start with makeing an instance of the Content 

  >>> from uvcsite.tests.simpleaddon import Contact

  >>> contact = Contact() 
  >>> contact 
  <uvcsite.tests.simpleaddon.Contact object at ...> 

Attributes
----------
There should be the two attributes from our IContent Schema

  >>> contact.name

  >>> contact.alter


Schema
------

  >>> contact.schema
  [<InterfaceClass uvcsite.tests.simpleaddon.IContact>] 


BaseClasses
-----------

  >>> from uvcsite.content.components import Content, ProductFolder

  >>> content = Content()
  >>> content.schema
  [<InterfaceClass dolmen.content.interfaces.IBaseContent>] 

  >>> print content.meta_type
  Content
