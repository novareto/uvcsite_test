Do a unit doctest test on the base.py 
=====================================

:Test-Layer: functional 

Let's first create an Object which inherits from Content:

   >>> from uvcsite import Content, name
   >>> from uvcsite import IContent
   >>> from megrok.icon import icon
   >>> from uvcsite.content.components import Icons


   >>> class Person(Content):
   ...    """ Base Class For Person """
   ...    name('Person')
   ...    icon('emblem-default', Icons)

Does Person implement the IContentType Interface?

   >>> IContent.implementedBy(Person)
   True

Now create an instance of Person and look for the
behaviour.

   >>> klaus = Person()
   >>> IContent.providedBy(klaus)
   True

   >>> print klaus.meta_type
   Person


Icons
-----

We have a default icon for all of our Content Objects..

   >>> from zope.site.hooks import setSite
   >>> root = getRootFolder()
   >>> setSite(root)


   >>> from megrok.icon import get_component_icon_url 
   >>> from zope.publisher.browser import TestRequest
   >>> request = TestRequest()

   >>> get_component_icon_url(Person, request)
   'http://127.0.0.1/++icon++icons/emblem-default'


Of course it should be possilbe to override the default one:

   >>> class CustomPerson(Content):
   ...    """ Base Class For Person """
   ...    name('Person')
   ...    icon('emblem-system', Icons)


   >>> get_component_icon_url(CustomPerson, request)
   'http://127.0.0.1/++icon++icons/emblem-system'
